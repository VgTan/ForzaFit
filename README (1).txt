Aplikasi ForzaFit
a. ForzaFit merupakan aplikasi gamifikasi dari sebuah aktivitas seperti olahraga untuk diet ataupun untuk menaikkan berat badan. Aktivitas ini ditemani oleh avatar hewan seperti beruang atau ayam yang dapat dipilih user dan dapat bersama-sama beraktivitas sampai ke target.

c. Fitur:
- Authentication
- tracking GPS
- workout list
- upload photo profile

Flow Aplikasi:
1. User akan melakukan login, namun apabila belum mempunyai akun, maka user akan melakukan signup dan mengisi data pribadi
2. Setelah semua data terisi, user akan diminta untuk memilih avatar yang mau digunakan untuk menemani mereka selama journey/perjalanan mereka sampai menggapai target
3. Setelah memilih avatar, user dipindahkan ke halaman homepage dimana user dapat melakukan penambahan aktivitas dan dapat memilih aktivitas mana yang mau dilakukan.
4. Setelah melakukan aktivitas, user akan mendapatkan exp sesuai dengan aktivitas yang dikerjakan
5. User juga dapat melihat progress perjalanannya pada profile yang dimana user juga dapat mengedit profile mereka dengan menambahkan/mengubah profile picture, dsb
